# docker-compose
Manage several containers at the same time. By default, there is a `docker-compose.yml` file. 


## Installation
- `sudo apt-get install python-pip`
- `sudo pip install docker-compose`


## Build and Run
- `docker-compose up`
- `docker-compose ls`
- `docker-compose down/stop`


## Static Web
- `docker-compose up`
- `http://localhost:5000`: check
